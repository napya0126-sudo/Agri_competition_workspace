# Agri Competition Workspace

EU Agri-Hackathon 2026 への応募を検討する過程で発覚した課題と、次のアクションを管理するリポジトリです。

## 経緯

- 応募先: [EU Agri-Hackathon 2026](https://agriculture.ec.europa.eu/overview-vision-agriculture-food/digitalisation/eu-agri-hackathon_en)(2026年10月16〜18日、オンライン開催、DG AGRI主催)
- Applicants Kit を確認した結果、**EU加盟国の市民権が必須(Pass/Fail)** の応募条件であることが判明。選考通過者にはパスポート/IDカードによる本人確認もあるため、非EU国籍での応募は事実上不可能。
- 応募者本人の背景: 獣医向け電子カルテ(PIMS)開発会社に勤務、実家は酪農家、大学時代は馬術部、新卒は酪農機械の輸入商社。動物領域×ハードウェア×ソフトウェアの知見が強み。

## 現在の方針

1. **主催者への正直な問い合わせ**(`correspondence/organizer-inquiry-draft.md`)— 非EU国籍でも参加できる形(オブザーバー、非公式アドバイザー等)がないか確認する。
2. **代替コンペの調査**(`research/alternative-competitions.md`)— 国籍不問のグローバル/日本のアグリテック系ハッカソン・コンペを探し、同じ強みを活かせる応募先を探す。
3. 上記の結果を踏まえて、実際に提出する応募文をこのリポジトリで作成する。

## ディレクトリ構成

- `correspondence/` — 主催者とのやり取りの下書き・記録
- `research/` — 代替コンペ調査メモ
- `applications/` — 実際に提出する応募文の下書き(将来追加)
